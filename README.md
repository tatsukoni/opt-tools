# opt-tools
opt tools.
Used to configure Vs Code.

## How to use
Before you can register an alias, you need to pass it through.

- ads: add_setting.sh
- dps: display_setting.sh
- dls: delete_setting.sh
- rgt: register_tag.sh

## set setting.conf
Before you start using this system, you need to configure the configuration.
Target file is /etc/setting.conf.<br>

- SETTING_PATH is path of editor configuration file.
- SETTING_FILE_NAME is path of editor tools setting file.
- REGISTER_TAG_FILE_NAME is path of editor tools register tags name file.
